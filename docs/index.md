# Stock Market SDK – Documentation

Welcome to the official documentation for the **Stock Market SDK project**, which provides:

- A RESTful stock market API hosted in the cloud (Flask + MongoDB Atlas)
- A fully functional Android SDK (available via JitPack)
- A demo Android app showcasing SDK features
- An admin analytics portal for usage tracking

## Table of Contents

- [Android SDK Usage](sdk_usage.md)
- [API Endpoints](api_endpoints.md)
- [Example Android App](example_app.md)

## Getting Started

1. Add the SDK to your Android project (see [SDK Usage](sdk_usage.md))
2. Use the base API URL: https://stockmarketapi-qr65.onrender.com/
3. Explore the demo app or integrate your own features!
